In order to get your development environment up and running, please follow the instructions of  \Cref{sec: Installation}. However, do not install Eclipse as described there, but instead install the following:

\begin{itemize}
\item Eclipse IDE for Java and DSL Developers (e.g. Version Luna, which contains Xtend and Xtext 2.7.3)
\item Eclipse features
\begin{itemize}
	\item GlassFish Tools for Luna
	\item Eclipse Java EE Developer Tools
	\item Eclipse Java Web Developer Tools
	\item Data Tools Platform Extender SDK
	\item JSF Tools
\end{itemize}

\end{itemize}
